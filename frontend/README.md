# 🌌 AtmoLens

### **Automated ECCC Synoptic Map Enhancement**
> Grayscale Environment Canada charts transformed into high-contrast, color-enhanced weather narrations — automatically, every 30 minutes.

---

## 🎨 Aesthetic DNA: "Bit Depth"
AtmoLens is built with a custom **Bit Depth** design system, blending traditional paper-and-ink textures with modern, obsidian-grade digital interfaces.

### 🌓 The Two Realities
| **Scrapbook Mode (Light)** | **Obsidian Mode (Dark)** |
| :--- | :--- |
| ![Light Mode](public/assets/docs/screenshots/about-light.png) | ![Dark Mode](public/assets/docs/screenshots/about-dark.png) |
| Warm, tactile "Antique White" journal aesthetic with semi-transparent jagged tape and handcrafted shadows. | Deep, high-contrast digital log with cyan-glow accents and nocturnal clarity. |

---

## ✨ Key Features
- **🖌️ Automated Normalization**: Real-time OpenCV-driven extraction of meteorological data from static ECCC synoptic charts.
- **📔 Storytelling UI**: A unique "Notebook" narrative for project documentation, featuring skeuomorphic binding and realistic scrapbook elements.
- **⚡ Obsidian Performance**: Optimized for ultra-low latency weather data retrieval and rendering.
- **🕵️ Built-in QA/QC**: Interactive dashboard for verifying data integrity across multiple meteorological layers.

## 🚀 Getting Started

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/thaparSAAB14/AtmoLens.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Deployment
Automatic tracking and performance metrics are handled via the **Vercel Analytics & Speed Insights** integration.
